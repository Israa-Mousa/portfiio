# ✏️ قائمة التخصيص - تخصيص البورتفيليو الخاص بك

## 📝 الخطوات الأساسية

اتبع هذه القائمة لتخصيص البورتفيليو بمعلوماتك الشخصية:

### 1. المعلومات الأساسية

- [ ] **الاسم:** عدّل "إسراء موسى" إلى اسمك
- [ ] **البريد الإلكتروني:** غيّر `israa@example.com`
- [ ] **الهاتف:** حدّث رقم الهاتف
- [ ] **الموقع:** غيّر "الأردن" إلى موقعك

**مكان التعديل:** `index.html` في قسم Contact

```html
<!-- ابحث عن هذا الجزء وعدّله -->
<p>israa@example.com</p>
<p>+962 XX XXXX XXX</p>
<p>الأردن</p>
```

---

### 2. روابط وسائل التواصل

- [ ] **LinkedIn:** حدّث رابط LinkedIn
- [ ] **GitHub:** أضف رابط GitHub
- [ ] **Facebook:** أضف رابطك (اختياري)
- [ ] **Twitter:** أضف رابطك (اختياري)

**مكان التعديل:** `index.html` في قسم Social

```html
<!-- ابحث عن هذا الجزء -->
<a href="https://www.linkedin.com/in/your-profile" target="_blank">
    <i class="fab fa-linkedin"></i>
</a>
<a href="https://github.com/your-username" target="_blank">
    <i class="fab fa-github"></i>
</a>
```

---

### 3. قسم "عني" (About)

- [ ] **الوصف الشخصي:** عدّل الفقرات لتعكس خبرتك
- [ ] **عدد المشاريع:** غيّر "10+"
- [ ] **سنوات الخبرة:** غيّر "5+"
- [ ] **رضا العملاء:** غيّر النسبة إذا لزم

**مكان التعديل:** `index.html` في قسم About

```html
<p>
    اكتب عن نفسك هنا...
    تحدث عن مهاراتك وخبرتك وشغفك بتطوير الويب
</p>
```

---

### 4. مهاراتك

تعديل المهارات:

- [ ] **المهارة 1:** عدّل اسم ووصف كل مهارة
- [ ] **يمكنك إضافة:** مهارات إضافية
- [ ] **أيكونات:** غيّر الأيقونات إذا أردت

**مكان التعديل:** `index.html` في قسم Skills

```html
<div class="skill-card">
    <div class="skill-icon">
        <i class="fab fa-html5"></i>  <!-- غيّر الأيقونة -->
    </div>
    <h3>HTML5</h3>  <!-- عدّل الاسم -->
    <p>بناء هياكل ويب قوية وسيمانتية</p>  <!-- عدّل الوصف -->
</div>
```

**أيقونات متاحة:**
- Languages: `fa-html5`, `fa-css3-alt`, `fa-js-square`
- Frameworks: `fa-react`, `fa-angular`, `fa-vuejs`
- Tools: `fa-git-alt`, `fa-database`, `fa-code`

[اعرض جميع الأيقونات](https://fontawesome.com/icons)

### 5. مشاريعك

- [ ] **المشروع 1:** غيّر الاسم والوصف
- [ ] **الصورة:** أضف صور حقيقية
- [ ] **التقنيات:** حدّث التقنيات المستخدمة
- [ ] **الرابط:** أضف رابط المشروع

**مكان التعديل:** `index.html` في قسم Projects

```html
<div class="project-card">
    <div class="project-image">
        <!-- غيّر الرابط إلى صورة حقيقية -->
        <img src="https://your-image-url.com/image.jpg" alt="اسم المشروع">
    </div>
    <div class="project-content">
        <h3>اسم المشروع</h3>
        <p>وصف المشروع هنا</p>
        <div class="project-tech">
            <span>تقنية 1</span>
            <span>تقنية 2</span>
        </div>
    </div>
</div>
```

### كيفية إضافة صور حقيقية:

**الطريقة 1: صور مجانية**
- [Unsplash](https://unsplash.com)
- [Pexels](https://pexels.com)
- [Pixabay](https://pixabay.com)

**الطريقة 2: رفع صور شخصية**
1. قم بإنشاء مجلد `images` في مجلد البورتفيليو
2. ضع الصور فيه
3. غيّر المسار: `<img src="images/project1.jpg">`

### 6. الألوان والتصميم

### تخصيص الألوان

**مكان التعديل:** `styles.css` - الأسطر الأولى

```css
:root {
    --primary-color: #667eea;        /* اللون الأساسي - البنفسجي */
    --secondary-color: #764ba2;      /* اللون الثانوي */
    --text-dark: #1a202c;            /* لون النص الداكن */
    --text-light: #6b7280;           /* لون النص الفاتح */
    --bg-light: #f8f9fa;             /* لون الخلفية الفاتح */
    --bg-white: #ffffff;             /* اللون الأبيض */
}
```

**أمثلة على ألوان:**
```css
/* أحمر */
--primary-color: #ef4444;
--secondary-color: #dc2626;

/* أزرق */
--primary-color: #3b82f6;
--secondary-color: #1e40af;

/* أخضر */
--primary-color: #10b981;
--secondary-color: #059669;

/* برتقالي */
--primary-color: #f97316;
--secondary-color: #ea580c;
```

### 7. الخطوط والأحجام

```css
/* في styles.css */

```
