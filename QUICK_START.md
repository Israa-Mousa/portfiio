## 🖥️ خوادم محلية (اختياري)

إذا أردت استخدام خادم محلي (موصى به):

### Python (الطريقة الأسهل)

**على Windows:**
```bash
# افتح Command Prompt أو PowerShell
# ثم انتقل إلى المجلد:
cd C:\Users\israa mazar3a\Documents\portfiio

# شغّل الخادم:
python -m http.server 8000
```

**أو (Python 2):**
```bash
python -m SimpleHTTPServer 8000
```

**افتح في المتصفح:**
```
http://localhost:8000
```

---

### Node.js (إذا كان لديك)

```bash
# تثبيت http-server
npm install -g http-server

# تشغيل الخادم
http-server

# افتح المتصفح
http://localhost:8080
```

---

### VS Code Live Server

**الأسهل والأفضل:**

1. **تثبيت VS Code ** من [code.visualstudio.com](https://code.visualstudio.com)

2. **تثبيت Extension:**
   - اضغط `Ctrl + Shift + X`
   - ابحث عن "Live Server"
   - اضغط Install

3. **تشغيل الموقع:**
   - افتح `index.html`
   - انقر بزر الفأرة الأيمن
   - اختر "Open with Live Server"

4. ✨ الموقع سيفتح تلقائياً!

## 🐛 حل المشاكل الشائعة

### ❌ "لا يمكن العثور على الملف"

**الحل:**
- تأكد من أن جميع الملفات في نفس المجلد
- تأكد من أسماء الملفات صحيحة (أحرف صغيرة)

### ❌ "الصور لا تظهر"

**الحل:**
- استخدم صور من الإنترنت مع روابط كاملة
- أو ضع الصور في مجلد `images`

### ❌ "الأيقونات لا تظهر"

**الحل:**
- تحقق من الاتصال بالإنترنت (الأيقونات من CDN)
- انتظر تحميل الصفحة بالكامل

### ❌ "الموقع بطيء"

**الحل:**
- استخدم صور أصغر وأسرع
- استخدم الخادم المحلي
- امسح cache المتصفح (Ctrl + Shift + Delete)
